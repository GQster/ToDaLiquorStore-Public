# 🥃 To Da Liquor Store! - Public Config

This repository contains the dynamic configuration for the **To Da Liquor Store!** Android app. 

### 📡 Remote Configuration (`remote_config.yaml`)

This file allows the app to update its search parameters and UI strings without requiring a full update in the Play Store. It is used for:

- **Holiday Overrides**: Changing the app to point to other points of interest (e.g., Gyms for April Fools).
- **Search Optimization**: Tuning Overpass and Nominatim search queries.
- **Force Updates**: Pushing mandatory update notifications if a minimum version is required.

---
*Note: The source code for the app itself is maintained in a private repository.*
